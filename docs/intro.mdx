## What is lux?
Lux is a cross-platform proxy client that is open-sourced.
It supports both the Windows and macOS operating systems, can run at the network layer.

:::note
Lux is not intended to replace clash, and it does not aim to do so.
:::


## Feature

* Support all proxy protocols from clash.
* Support Tun and System Proxy
* Light and low memory usage.
* Built-in proxy rules.
* GUI-first, zero-conf, easy to use.
